# CLAUDE.md

This file guides AI assistants (Claude and others) working in this repository.

## Documentation standard

The documentation writing standard for this repository lives in [CONTRIBUTING.md](./CONTRIBUTING.md). It is the single, tool-agnostic source of truth for brand and product naming, terminology, front matter, heading hierarchy, tags, internal links, screenshots, redirects, and discoverability.

When writing or editing documentation, follow and enforce every rule in [CONTRIBUTING.md](./CONTRIBUTING.md). Do not duplicate those rules here; if a rule needs to change, change it in `CONTRIBUTING.md` so all contributors and tools stay in sync.

## Working in this repository

- This is a [Docusaurus](https://docusaurus.io/) site. Documentation pages live under `docs/`.
- Run `yarn` to install dependencies, `yarn start` for a local dev server, and `yarn build` to produce the static build.
- Before opening a pull request, review your changes against [CONTRIBUTING.md](./CONTRIBUTING.md).
